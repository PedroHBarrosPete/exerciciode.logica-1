Exercício de Lógica- Gerador de Recibos
Código escrito baseado na proposta do execício 10 da lista
de exercícios sobrevariáveis da matéria de Algorítimos e Lógicade Programação.

Descrição: Crie um algoritmo completo para gerar um recibo de 
compra. O sistema deve: 1. Ler o nome do cliente. 2. Ler o nome
de 3 produtos e o preço de cada um. 3. Calcular o subtotal (soma
dos 3 preços). 4. Aplicar desconto de 10% se o cliente for membro
(leia S ou N). 5. Calcular o total final. 6. Exibir um recibo formatado
com: nome do cliente, os 3 produtos com preços, subtotal, desconto
(se houver) e total.Obs.: neste exercício, use apenas atribuição com
operador ternário implícito — ainda sem se/então. Para o desconto, 
sempre calcule: desconto ← subtotal * 0.10, e some ou não ao final 
conforme quiser testar.

Foi adaptada a proposta do exercício para um código em Java.
